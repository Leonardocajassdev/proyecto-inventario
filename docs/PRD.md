# Product Requirements Document

## Objetivo
Desarrollar un sistema de inventario que permita gestionar productos y movimientos de stock.

## Requisitos

- Registro de productos
- Control de inventario
- Movimientos de entrada y salida
- Validación de stock mínimo
- Interfaz simple en React

## Reglas de negocio

- No permitir stock negativo
- No permitir bajar de stock mínimo
- Mostrar alertas visuales
